## 工具

```
    root
    │      
    └───html
    │   └───image
    │   ├── comimage.html  (前端对图片的压缩)
    │   ├── scanning.html  (扫描动画)
    │   ├── iscroll.html   (滚动条相关)
    │   ├── video.html     (视频播放)
    │    
    └───node
    │   ├── email.js        (发送邮箱)
    │   ├── fromjsontoexcel.js　　　（json转excel）
    │   ├── mysql.js        (mysql)
    │   ├── pstgresql.js　　 (pstgresql)
    │   ├── redis.js        (redis)
    │   ├── request.js      (http请求)
    │   ├── saveImage.js    (保存图片)
    │   ├── socket.js       (一个完整的socket示例)
    │           
    └───python
        ├── phone.py        (随机生成电话,并保存为csv文件)
```
